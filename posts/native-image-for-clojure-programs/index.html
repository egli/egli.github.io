Native Image for Clojure Programs

Posted on 2019-10-31 One min read

The release helper program to create liblouis releases would be a typical shell script: Move some files around, replace some text, etc. As always with shell scripting what starts out easy quickly becomes unwieldy as the requirements grow.

In particular there is one requirement that is a bit tricky. We have a regular release schedule four times a year in ISO week 10, 23, 36 and 49. To calculate this is basically impossible with standard shell commands. In clojure this can be solved very elegantly with an infinite sequence.

Now since this is basically used as a script at release time it should start quickly. One of the promises of GraalVM is that it can make JVM-based programs start very fast. Why not try it?
