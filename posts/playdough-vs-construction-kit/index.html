Playdough vs Construction Kit

Posted on 2014-12-19 One min read

Clojure code is so maleable. You can just build anything you want. The flip side is that many of the things you want you will have to build yourself.

With the construction kit it is really easy to build the 3 models that you can build with it. If all you want is one of the 3 models then you're fine. If you want something else though, you'll find yourself scouring the internets and the source code of the construction kit and end up hacking around the framework.

Compare this to the playdough approach where anything can be formed from simple sexps.
