A Different Kind of Notebook

Posted on 2019-11-15 2 mins read

Netflix just released Polynote, a a different kind of notebook as they say. Unlike Jupyter notebooks it supports, wait for it, mixing multiple languages in one notebook.

But wait, wasn't this possible all along with Org Babel? Just look at the Babel web site and you'll know that is has been possible for a long time.

This reminds me of a nifty notebook that I wrote in June 2016 to trace license compliance of liblouis tables. It mixes shell code to fetch the relevant data and Clojure code to mangle it.

The shell code is simple 

#+NAME: Other-license-tables
#+BEGIN_SRC shell
cd ~/src/liblouis/tables
licensecheck --copyright --check='\.(ctb|utb|cti|uti|dis)' --encoding utf8 --machine * | grep -v 'LGPL (v2.1 or later)' | grep -v 'LGPL (v3 or later)'
#+END_SRC

This will insert the data in your notebook. In this case you'll get a table of all files with unknown license 

#+RESULTS: Other-license-tables
| ca-chardefs.cti | UNKNOWN | 2010 DocArch <http:www.docarch.be>                                                              |
| ca-g1.ctb       | UNKNOWN | 2010 DocArch <http:www.docarch.be>                                                              |
| es-new.dis      | UNKNOWN | 2010 DocArch <http:www.docarch.be>                                                              |
| es-old.dis      | UNKNOWN | 2010 DocArch <http:www.docarch.be>                                                              |
| ethio-g1.ctb    | UNKNOWN | 2012, 2015 Mesar Hameed <mesar.hameed@gmail.com> / 2012 Dr. Tamru E. Belay <tamru@sympatico.ca> |
| zh-hk.ctb       | UNKNOWN | 2009 KM Yuen, Keny Yuen, Alex Ho and Eric Yip / 2009 JJB Software, Inc. www.jjb-software.com    |

If I come across as an old, know-it-all grump then probably because I am. Of course you cannot really compare Jupyter notebooks and Org babel. The usability is just one thing where they differ. But the claim to difference just because you support mixing multiple languages in one notebook seems a little bit exagerated.
