Code Is a Liability

Posted on 2019-10-30 One min read

I don't know why it took so long, but word finally got to me: Features are an asset, Code is a liability

A few years ago you put this wonderful code in production. It was like launching this great steamer on the Mississipi. It had all the latest in steam engine technology and all the new-fangled toys-du-jour. Well, actually you wanted to use the ruby-coloured engine but your company didn't have that in their stack, so you used a different brand engine.

But alas quite a bit of water has come down that river and steam engines aren't the latest fad anymore. In fact you can't even get parts for the engine anymore let alone find anyone wanting to maintain that thing.

Meanwhile you're users happily use the ship to get places. Sometimes they complain about it being a little slow but maybe they like the old-timer charme that it evaporates.

Down in the machine room you're seemingly the only one that is bothered by the creakiness of this whole operation. Many more cabins were bolted onto this ship on its long journey. Many an obstacle came floating down and sometimes crashed into your boat. In fact the river hardly seems what it used to be. The boat barely stays afloat.
